# Maximum Security Password Generator

A secure password generator built to create strong, random passwords for protecting online accounts and sensitive information.

This project allows users to generate highly secure passwords with customizable options such as length and character types.

## Features

- Generate strong random passwords
- Adjustable password length
- Uppercase letters (A–Z)
- Lowercase letters (a–z)
- Numbers (0–9)
- Special characters (!@#$%^&*)
- Copy password to clipboard
- Strength indicator
- Responsive user interface
- Prevent empty or weak password generation

## Security Principles

This project follows common password security practices:

- Uses cryptographically secure random generation (where supported)
- Avoids predictable patterns
- Supports long password lengths
- Encourages unique passwords for every account
- Generates balanced character combinations

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/password-generator.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

## How to Use

1. Click **Generate Password**
2. Copy and use your password securely

## Security Recommendations

- Use passwords with at least **16–20 characters**
- Use a different password for every account
- Store passwords in a password manager
- Enable multi-factor authentication (MFA)
- Never share generated passwords
